## Install anywhere

Optionally set the starship BIN_DIR (example):

```sh
export STARSHIP_BIN_DIR="${HOME}/.local/bin"
```

```sh
curl -fsSL https://raw.githubusercontent.com/andreswebs/features/refs/heads/main/src/zsh-starship/install.sh | sh
```
